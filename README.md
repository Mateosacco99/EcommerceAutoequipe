# AutoEquipe Ecommerce

- Proyecto final para el curso de React


### Instalacion
- clonar repo

- cd a carpeta del repo

- npm install

> Es necesario tener Node v22.20.0

### Correr proyecto
- npm run dev

### correr local para ver en mobile

- npm run dev -- --host

### Librerias

- [react-icons] Libreria de Iconos

- [react-router-dom] : Navegacion

- [sass] : Styling

- [firebase] : Base de Datos
